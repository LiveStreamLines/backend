Camera Widget Iframe Examples

How to embed the camera widget:

Basic embedding:

<iframe 
    src="https://lsl-platform.com/backend/public/camera-widget.html?projectId=YOUR_PROJECT&cameraId=YOUR_CAMERA&token=YOUR_TOKEN" 
    width="600" 
    height="500" 
    frameborder="0"
    allowfullscreen>
</iframe>

URL Parameters:

  • projectId - Your project identifier
  • cameraId - Your camera identifier
  • token - Your authentication token

Configuration Options:

Option 1: URL Parameters (Recommended)

Pass configuration through URL query parameters as shown above.

Option 2: Edit HTML File

Update the default values in camera-widget.html:

  • projectId: Your project ID
  • cameraId: Your camera ID
  • authToken: Your authentication token
